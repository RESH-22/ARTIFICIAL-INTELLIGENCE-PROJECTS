# ▶️ How To Run

1️⃣ Clone the repository

```
git clone https://github.com/RESH-22/ARTIFICIAL-INTELLIGENCE-PROJECTS.git
```

2️⃣ Navigate to the folder

```
cd ARTIFICIAL-INTELLIGENCE-PROJECTS
```

3️⃣ Open any project folder

Example:

```
cd Face\ Detection
```

4️⃣ Run the Python file

```
python face_detection.py
```
